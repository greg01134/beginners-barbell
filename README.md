# The Beginner's Barbell 🏋️‍♂️

**A simple, non-intimidating guide to starting your strength training journey.**

## 📌 Project Overview
The gym environment can often feel overwhelming for beginners. "The Beginner's Barbell" is a prototype two-page website designed to lower the barrier to entry for strength training. It introduces newcomers to the foundational "Big 3" lifts (Squat, Bench Press, and Deadlift) in a welcoming, easy-to-understand format. 

This repository serves as the complete project package for **Project 03**, demonstrating both technical web development skills and professional project management documentation.
