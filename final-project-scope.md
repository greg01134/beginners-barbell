# Project Scope Statement: Beginner's Guide to Strength Training
Project Name: The Beginner's Barbell: Overcoming Gym Intimidation

Project Manager: Greg Gaines

**1. Project Justification**
The gym environment can be overwhelming and intimidating for beginners, often preventing people from starting their fitness journey. This project aims to lower the barrier to entry by providing a simple, clear, and non-intimidating resource that focuses on the three most foundational lifts. By narrowing the focus, the site helps novices build confidence and basic knowledge without being flooded by complex information.

**2. Product Scope Description**
The project will result in a two-page "prototype" website designed with a clean, welcoming aesthetic.

Page 1 (Home - index.html): An introduction to the "Big 3" lifts (Squat, Bench Press, and Deadlift). It will include high-level descriptions of why these exercises are important and basic tips for getting started.

Page 2 (About/Contact - about.html): This page will detail my training philosophy as a coach who has been training clients for almost 11 years. (emphasizing consistency and simplicity) and provide contact information for users who have follow-up questions.

Styling: A single external CSS file (style.css) will be used to ensure a cohesive look and feel across both pages.

**3. Project Deliverables**
To complete this project, the following items will be produced and submitted:

A public GitHub repository housing all project files.

final-project-scope.md: This formal scope statement (Assignment 03).

index.html: The structured homepage content.

about.html: The secondary page for philosophy and contact.

style.css: The external stylesheet for site-wide design.

README.md: A professional landing page for the GitHub repository explaining the project purpose.

**4. Project Acceptance Criteria**
The project will be considered successful if:

The website is live and accessible through GitHub Pages.

Navigation between the Home and About pages is functional with no broken links.

The content is written in a beginner-friendly tone that addresses "gym-timidation."

The CSS is applied correctly to both HTML files, demonstrating a consistent design.

The code is semantic, valid, and contains professional comments.

**5. Project Exclusions**
To maintain a manageable scope for this introductory course, the following are not included in this project:

Video tutorials or large multimedia embeds.

Backend functionality (e.g., no functional database, user accounts, or workout logs).

E-commerce elements (e.g., selling plans or supplements).

Advanced training techniques, accessory lifts, or complex periodization programs.

Interactive JavaScript elements or complex forms.
